# vue3-electron

it's a desktop paint/photoshop-like project on vue 3 + electron

Was made as a homework for university

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
